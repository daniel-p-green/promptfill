# @brand-kit/stripe

Portable, opinionated Remotion brand kit.

## Includes
- Tokens, fonts, and brand provider
- Typed copy schema with zod validation
- Typography/layout primitives with hard constraints
- Script-driven reel compositions for 9:16 and 16:9

## Usage

```ts
import { ScriptDrivenReel, scriptDurationInFrames, stripeExampleScript } from "@brand-kit/stripe";
```

## Public assets

Copy `src/brand/logo.svg` to your project at `public/brand/stripe/logo.svg`.
